    替换文本

    先用notepad++去替换需要更换的文本

    查找：需要替换的文本左侧 → [[[ 需要替换的文本右侧 → ,,,

    提取文本： Excel中使用公式 =MID(S2,FIND("[[[",S2)+3,FIND(",,,",S2)-3-FIND("[[[",S2))

    提取位置： Excel中使用公式 =MID(H2,FIND("[",H2),FIND("]",H2)+1-FIND("[",H2))

    强制合并： Excel中使用公式 =R2&S2
